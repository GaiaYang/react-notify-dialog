# 組件自定義說明

## 替換通知 UI

依照需求替換以下組件即可更換組件樣式

| 組件名稱          | 組件說明 | 原生組件 |
| ----------------- | -------- | -------- |
| Dialog            | 容器     | `dialog` |
| DialogContent     | 內容     | `div`    |
| DialogTitle       | 標題     | `h2`     |
| DialogButton      | 按鈕     | `button` |
| DialogDescription | 訊息     | `p`      |
| DialogFooter      | 按鈕列   | `div`    |
